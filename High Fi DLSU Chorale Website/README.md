# DLSU Chorale Website

High-fidelity official website and portal for the De La Salle University Chorale, featuring:

## Features

✅ **Landing Page** - Elegant split-screen design with hero copy and background imagery
✅ **Dark Mode** - Full light/dark theme support with toggle
✅ **Member Portal** - Dashboard with attendance tracking, excuse management, performance sign-ups, and fee management
✅ **Admin Console** - Comprehensive admin dashboard for managing excuses, attendance, performances, and finances
✅ **Design System** - Complete UI component library (Cards, Buttons, Chips, Avatars, Icons)
✅ **Typography** - Cormorant Garamond (serif), Inter Tight (sans-serif), JetBrains Mono (monospace)

## Tech Stack

- **Framework**: React + TypeScript
- **Styling**: Tailwind CSS v4
- **Icons**: Lucide React
- **Fonts**: Google Fonts (Cormorant Garamond, Inter Tight, JetBrains Mono)

## Design Features

- Elegant minimalistic design
- DLSU Chorale branding and color palette
- Green accent colors (#115E2B)
- Responsive layouts
- Smooth transitions and animations

## Navigation

### Landing Page
- Enter as Member → Member Dashboard
- Enter as Admin → Admin Console
- Dark mode toggle (top right)

### Member Portal
- Home - Dashboard with recent attendance, upcoming performances
- My Attendance - Full attendance history with calendar view
- Excuse Approvals - File and track excuse requests
- Performances - Browse and sign up for performances
- Fees & Payments - View and manage outstanding fees
- Announcements - Chorale announcements and updates
- Profile - Member profile and settings

### Admin Console
- Dashboard - Overview with pending excuses and statistics
- Attendance Overview - Section-level attendance tracking
- Excuse Approvals - Review and approve/decline excuses
- Performances - Manage events and rosters
- Fee Management - Track balances and payments
- Analytics - Reports and trends
- Members - Member directory and management
